자연어 코드 수정

복잡한 코드 문법을 몰라도 괜찮습니다. 원하는 변경사항을 일상 언어로 설명하면 AI가 정확히 이해하고 코드를 수정합니다.

  • ✓ 한국어로 자유롭게 요청
  • ✓ 맥락을 이해하는 스마트 AI
  • ✓ 복잡한 수정도 한 번에
  • ✓ 추가 질문으로 명확한 수정

요청

"상품 목록 페이지에서 가격이 10만원 이상인 상품들은 '프리미엄' 배지가 보이도록 해줘"

AI 응답

ProductList 컴포넌트를 수정했습니다. price >= 100000인 상품에 프리미엄 배지를 추가했습니다.

{price >= 100000 && <Badge type="premium">프리미엄</Badge>}

  • ✓ 원클릭 즉시 롤백
